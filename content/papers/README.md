# 📚 Papers Collection

이 폴더에는 읽고 정리한 모든 연구 논문들이 체계적으로 정리되어 있습니다.

## 📁 폴더 구조

- **by-year/**: 연도별 논문 분류
- **by-topic/**: 주제별 논문 분류
- **classics/**: 고전 논문들
- **recent/**: 최신 논문들

## 🏷️ 태그 시스템

- `#paper` - 모든 논문
- `#AI` - 인공지능
- `#ML` - 머신러닝
- `#CV` - 컴퓨터 비전
- `#NLP` - 자연어처리
- `#year-YYYY` - 연도별 분류
